Magma
==========

Magma is the sub-package of Lava providing the main components of Lava 
  
  - :ref:`Magma core <lava/lava.magma.core:lava.magma.core>`
    base-classes for :ref:`Processes <lava/lava.magma.core.process:lava.magma.core.process.process>`, :ref:`ProcessModels <lava/lava.magma.core.model:lava.magma.core.model.model>` and :ref:`SyncDomains <lava/lava.magma.core.sync:lava.magma.core.sync.domain>` as well as basic behavior and functionality
  - :ref:`Magma compiler <lava/lava.magma.compiler:lava.magma.compiler.compiler>`
    building the network and the communications channels
  - :ref:`Magma runtime <lava/lava.magma.runtime:lava.magma.runtime.runtime>`
    providing a control interface for the user
